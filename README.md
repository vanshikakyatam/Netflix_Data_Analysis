# Netflix Data Analysis Project

## Overview

This project analyzes Netflix content data to uncover trends in content type, production countries, release patterns, and audience ratings. The goal is to derive meaningful insights that can support content strategy and decision-making.

---

## Tools & Technologies

* Python
* Pandas
* Matplotlib
* Visual Studio Code

---

## Dataset

* Netflix Titles Dataset (CSV format)
* Contains information about movies and TV shows including type, country, release year, rating, and date added

## 🔍 Key Insights

* Movies dominate Netflix content compared to TV shows
* A few countries contribute the majority of content
* Content production has significantly increased over the years
* Most content is targeted toward mature audiences (TV-MA, TV-14)
